
# ContractGuard AI

AI-powered financial contract risk analyzer that helps users understand loan agreements and credit card terms by detecting risks, hidden fees, and complex clauses.

## Features
- Upload financial documents (PDF)
- Detect important clauses (interest, penalties, hidden fees)
- Risk classification (Low / Medium / High)
- Simple explanation of legal terms
- Evidence-based insights

## Tech Stack
- Frontend: React.js
- Backend: FastAPI
- AI: OpenAI GPT
- Vector DB: FAISS (optional)
